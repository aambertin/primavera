(static) @Schema(jsonschema)

Source:
  • validations.js, line 30

Adds static and dynamic $validate methods to a given class/object. The $validate methods will take care of validating the object against the provided JSON schema.

Parameters:
Name Type Description
jsonschema object

the json schema that should apply to instances of this object.

(static) @ValidateSchema()

Source:
  • validations.js, line 105

Validate arguments of a method against json-schemas before allowing its execution.

Example
class MyClass {

    // validate arguments with a json schema...
    @ValidateSchema(SomeSchemaForArg1, SomeSchemaForArg2)
    async myMethod(arg1, arg2) {
        // ...
    }

    // or apply multiple schemas to one argument...
    @ValidateSchema([Arg1Schema1, Arg1Schema2], SomeSchemaForArg2)
    async myOtherMethod(arg1, arg2) {
        // ...
    }
    
}
